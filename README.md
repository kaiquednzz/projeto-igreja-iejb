# ⛪ IEJB — Site Oficial

Este projeto é o desenvolvimento de um website para a Igreja IEJB, com o objetivo de fortalecer sua presença digital e facilitar o acesso de membros e visitantes às principais informações da igreja.

Mais do que um projeto de estudo, esta aplicação foi pensada com potencial real de uso, podendo evoluir para uma plataforma completa no futuro.

---

## 🔗 Acesse o projeto

Você pode visualizar o site online através do botão abaixo:

👉 <a href="https://kaiquednzz.github.io/projeto-igreja-iejb/">Veja o projeto.</a>

---

## 🌐 Visão geral

O site foi projetado para ser simples, direto e acessível, priorizando a experiência do usuário. A navegação é fluida, com foco em apresentar conteúdos essenciais de forma clara e organizada.

Entre os principais objetivos:

* Tornar a igreja mais acessível online
* Melhorar a comunicação com visitantes
* Centralizar informações importantes
* Criar uma base para futuras funcionalidades

---

## ⚙️ Tecnologias

Este projeto foi desenvolvido utilizando tecnologias fundamentais do desenvolvimento web:

* HTML5
* CSS3
* JavaScript

## ✨ Funcionalidades

* Estrutura de navegação por seções
